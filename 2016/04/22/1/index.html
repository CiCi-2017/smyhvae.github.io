（1）链接的配置。链接放置在 social 字段下，其格式是 item label: Link，一行一个链接。举例：

（2）设定链接的图标。对应的字段是 menu_icons。此设定格式是 item name: icon name，其中 item label 与上一步所配置的链接的文本，icon name 是 Font Awesome 图标的 名字。而 enable 可用于控制是否显示图标，你可以设置成 false 来去掉图标。
